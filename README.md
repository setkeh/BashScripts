DropBox_Screens Usage
---------------------
dropbox_screens <args>
        
       ` --help =    Show this Help.`
       ` win    =    Select A Window Border With the Mouse.`
       ` scr    =    Take Screenshot of Entire Screen.`
       ` area   =    MultiHead Display Support (1 Shot each Screen).`
